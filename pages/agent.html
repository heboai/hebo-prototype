I'm trying to understand the difference between Core LLM and Condense LLM. Can you explain?
The Core LLM is the main model that generates responses and makes decisions about using tools. Think of it as the brain of your agent. The Condense LLM, on the other hand, is specifically used to process and simplify conversation history into concise queries for better context retrieval.
That makes sense. So for simpler use cases, could I use a lighter model for the Condense LLM?
Yes, exactly! For the Condense LLM, you could use a lighter model like GPT-4o mini or claude-3-5-haiku since the task is more straightforward. This would be more cost-efficient while still maintaining good performance.
